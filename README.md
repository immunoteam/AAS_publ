# Five dominant amino acid substitution signatures shape tumour immunity — figure code

This repository contains code for generating the **main figures** in:

> **Juhász et al.** *Five dominant amino acid substitution signatures shape tumour immunity*.

## What’s in this repo

- Figure-generation scripts used to reproduce the **main manuscript figures**.

> Note: Large files and raw input datasets are excluded from version control and can be downloaded from https://zenodo.org/records/17962590
